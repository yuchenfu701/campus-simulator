# 🚀 快速启动指南

## 📋 前置要求

1. **Node.js** - 下载安装：https://nodejs.org/
2. **MongoDB Compass** - 已安装并运行
3. **浏览器** - Chrome/Edge推荐

---

## ⚡ 快速启动（Windows）

### 方法1：使用启动脚本（推荐）

1. 双击运行 `启动服务器.bat`
2. 等待服务器启动完成
3. 打开浏览器访问前端页面

### 方法2：手动启动

```bash
# 1. 进入server目录
cd server

# 2. 安装依赖（首次运行）
npm install

# 3. 启动服务器
npm start
```

---

## 🔧 MongoDB配置

### 1. 启动MongoDB Compass

1. 打开 MongoDB Compass
2. 连接到本地 MongoDB（默认：`mongodb://localhost:27017`）
3. 如果连接失败，检查MongoDB服务是否运行

### 2. 创建数据库

MongoDB Compass会自动创建数据库，无需手动创建。

如果需要手动创建：
1. 点击"Create Database"
2. 数据库名：`campus_simulator`
3. 集合名：留空（会自动创建）

## 🐛 常见问题

### 问题1：端口被占用

**错误信息：** `Error: listen EADDRINUSE: address already in use :::3000`

**解决方案：**
1. 关闭占用3000端口的程序
2. 或修改 `server/.env` 中的 `PORT` 值

### 问题2：MongoDB连接失败

**错误信息：** `MongoDB连接失败`

**解决方案：**
1. 确保MongoDB Compass已启动
2. 检查连接字符串：`mongodb://localhost:27017`
3. 确认MongoDB服务正在运行

### 问题3：依赖安装失败

**解决方案：**
1. 检查网络连接
2. 使用国内镜像：
   ```bash
   npm config set registry https://registry.npmmirror.com
   ```
3. 清除缓存后重试：
   ```bash
   npm cache clean --force
   npm install
   ```
