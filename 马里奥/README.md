# 精美宝可梦2D像素游戏 - 《梦》

## 项目概述
这是一个基于HTML5 Canvas和JavaScript开发的2D平面像素游戏，类似于马里奥的游戏机制，但融入了精美的宝可梦像素艺术和丰富的剧情。每个宝可梦都经过精心设计，色彩丰富、细节精美！

## 游戏背景故事
**宝可梦游戏**

你是否有过成为宝可梦大师的想法？你又是否想成为大家的救世主？公元3025年，神明降下神罚，将世间美好变成斗争的游戏，唯有拥有驯服精灵的天赋者才能完成救世。

而你，正好拥有这个天赋，和皮卡丘相遇并驯服后开始了救世的道路。过程中，你认识了四位朋友，均为拥有驯服精灵的天赋者，你们互帮互助，度过了一道道难关。

结局就是开始，你又是否完成了救世，还是将人民重新推入深渊？来《梦》揭晓故事的答案。

## 游戏特色
- 🎮 经典的2D平台跳跃游戏机制
- 🎨 精美的像素艺术宝可梦设计
- 🎯 宝可梦收集与训练系统
- 📖 丰富的剧情故事线
- ✨ 色彩丰富的像素艺术风格
- 🎵 沉浸式的背景音乐
- 👥 多个可交互的NPC角色
- 🌟 细致入微的像素细节

## 精美像素艺术特色
- 🎨 每个宝可梦都经过精心设计，具有独特的色彩搭配
- 🎯 皮卡丘的鲜明黄色和可爱腮红
- 🔥 小火龙的温暖橙色和黄色肚子
- 💧 杰尼龟的清新蓝色和白色外壳
- 🌿 妙蛙种子的自然绿色和深绿色斑点
- 🌸 梦幻的粉红色和神秘光芒
- 💜 超梦的深紫色和强大气场

## 技术栈
- HTML5 Canvas
- JavaScript (ES6+)
- CSS3
- 无需额外框架，开箱即用

## 如何运行
1. 在项目目录中打开 `index.html`
2. 或者使用本地服务器运行
