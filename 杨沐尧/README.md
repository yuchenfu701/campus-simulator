# 🏃 火柴人大冒险

一个精心制作的火柴人平台跳跃游戏，使用纯 HTML/CSS/JavaScript 开发，无需任何外部依赖。

## ✨ 游戏特色

### 核心玩法
- **平台跳跃**：流畅的物理引擎，真实的重力系统
- **星星收集**：收集所有星星完成关卡
- **敌人躲避**：小心红色的敌人火柴人！
- **陷阱规避**：地面上的尖刺会造成伤害

### 游戏系统
- ⭐ **多关卡系统**：3个精心设计的关卡
- 📈 **难度递增**：随关卡提升，敌人速度和游戏速度逐渐增加
- 💾 **最高分记录**：使用 localStorage 保存你的最高分
- 🎵 **音效系统**：跳跃、收集、受伤、通关等音效
- ❤️ **生命系统**：3条生命，受伤后有无敌时间
- 🎨 **精美UI**：渐变色、动画效果、毛玻璃效果

## 🎮 操作说明

### 基本操作
- **← → 或 A D**：左右移动
- **空格 或 ↑ 或 W**：跳跃
- **M**：静音切换

### 游戏目标
1. 收集关卡中所有的金色星星
2. 避开红色敌人和地面尖刺
3. 完成所有关卡获得最高分

## 🎯 得分规则

| 项目 | 分数 |
|------|------|
| 收集星星 | +10 |
| 完成关卡 | +50 |
| 通关所有关卡 | +200 |

## 🚀 运行方式

**超级简单！** 直接用浏览器（推荐 Chrome、Edge、Firefox）打开 `index.html` 文件即可开始游戏。

**无需安装任何环境或依赖！**

## 🛠️ 技术特点

- **纯原生技术栈**：HTML5 Canvas + CSS3 + ES6+
- **Web Audio API**：实时音效生成
- **localStorage**：本地数据持久化
- **Canvas 2D**：高性能图形渲染
- **面向对象设计**：清晰的类结构
- **游戏循环**：requestAnimationFrame 优化

## 👨‍💻 开发说明

本项目完全使用原生 JavaScript 开发，代码结构清晰，注释详细，适合初学者学习游戏开发。

每个类都有明确的职责：
- `StickMan`：玩家角色
- `Platform`：平台
- `Star`：星星
- `Enemy`：敌人
- `Trap`：陷阱

游戏循环使用 `requestAnimationFrame` 确保流畅的 60fps 运行。
