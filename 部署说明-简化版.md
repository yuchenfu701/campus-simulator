# 🚀 快速部署到公网 - 5分钟搞定

让所有人通过网址访问你的游戏，无需下载任何文件！

---

## 📋 最简单的方法：GitHub Pages（推荐）

### 步骤1：注册GitHub账号（1分钟）

1. 访问：https://github.com/
2. 点击 "Sign up" 注册
3. 验证邮箱

### 步骤2：创建仓库（1分钟）

1. 登录后，点击右上角 "+" → "New repository"
2. 仓库名：`campus-simulator`（随便起名）
3. **必须选择 Public**（公开）
4. **不要**勾选 "Initialize with README"
5. 点击 "Create repository"

### 步骤3：上传文件（2分钟）

**方法A：使用GitHub Desktop（最简单）**

1. 下载：https://desktop.github.com/
2. 安装并登录
3. 点击 "File" → "Add Local Repository"
4. 选择你的项目文件夹（`d:\编程\test`）
5. 输入提交信息："Initial commit"
6. 点击 "Commit to main"
7. 点击 "Publish repository"
8. 等待上传完成

**方法B：网页上传（无需安装）**

1. 在GitHub仓库页面，点击 "uploading an existing file"
2. 将项目文件夹中的所有文件拖拽上传
   - **注意**：不要上传 `node_modules` 文件夹
   - **注意**：不要上传 `.git` 文件夹
3. 输入提交信息："Initial commit"
4. 点击 "Commit changes"
5. 等待上传完成

### 步骤4：启用GitHub Pages（1分钟）

1. 在GitHub仓库页面，点击 "Settings"
2. 左侧菜单找到 "Pages"
3. Source: 选择 "Deploy from a branch"
4. Branch: 选择 "main"
5. Folder: 选择 "/ (root)"
6. 点击 "Save"

### 步骤5：访问你的网站（等待1-2分钟）

1. 等待几分钟，GitHub会部署你的网站
2. 访问：`https://你的用户名.github.io/campus-simulator`
3. 如果显示404，等待几分钟后刷新

**完成！** 🎉

## 🔧 其他部署方式

### Vercel（更快速）

1. 访问：https://vercel.com/
2. 使用GitHub登录
3. 点击 "Import Project"
4. 选择你的GitHub仓库
5. 点击 "Deploy"
6. 完成！

### Netlify（最简单）

1. 访问：https://www.netlify.com/
2. 注册账号
3. 将项目文件夹拖拽到页面
4. 完成！

## ❓ 常见问题

**Q: 上传后显示404？**
- 等待几分钟（GitHub需要时间部署）
- 刷新页面

**Q: 图片无法显示？**
- 检查文件路径是否正确
- 确保所有文件都已上传

**Q: 功能无法使用？**
- 检查浏览器控制台错误（F12）
- 确保后端服务器已部署（如果需要）
