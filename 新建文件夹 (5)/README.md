# KARDS - 二战卡牌游戏

基于KARDS核心机制设计的卡牌策略游戏，使用纯 HTML/CSS/JavaScript 实现。

## 核心机制

### 战场构成
- **三条战线**：
  - 双方各有一条支援阵线（最多4个单位，总部所在）
  - 中间是共享的前线（最多5个单位）

### 前线控制
- 同一时间只能由一方占领
- **只有控制了前线，地面单位才能攻击敌方支援阵线的单位或总部**
- 前线控制由双方前线单位数量决定

### 资源系统
- **指挥点 (K)**：每回合自动补满，上限每回合+1（默认上限12点）
- 用于：部署单位、使用指令、移动/攻击

### 兵种行动规则
- **步兵**：每次行动只能在移动或攻击中选一项，攻击距离1格
- **坦克**：同一回合可以各进行一次移动和一次攻击，攻击距离1格
- **炮兵**：行动逻辑同步兵，但可以从支援阵线直接攻击敌方支援阵线，且攻击时不会受到反击
- **战斗机**：属于空军。敌方轰炸机无法攻击有战斗机的战线上的其他单位
- **轰炸机**：属于空军。攻击时无反击伤害。若敌方战线有战斗机，则只能攻击战斗机且会受到反击

### 关键词系统
- **部署**：单位被放置到战场时立即触发
- **亡计**：单位被消灭时触发
- **闪击**：单位可以在部署的同一回合就进行移动或攻击
- **奋战**：单位在一个回合内可以攻击两次
- **伏击**：单位每回合第一次受到攻击时，会率先对攻击者造成伤害
- **压制**：单位无法移动或攻击，状态在拥有者的下个回合结束时解除
- **重甲 (X)**：被攻击时，受到的伤害减少X点（最多可叠加3层）
- **烟幕**：单位不能被敌方单位选为攻击目标，但在其移动或攻击后会失去此效果
- **免疫**：单位或总部不会受到任何伤害
- **动员**：单位在自己的回合开始时获得+1/+1，受到伤害后失去该效果
- **情报**：使用后可以查看对手手牌中指定数量的卡牌
- **协力**：使用此牌时，若回合开始时场上没有相同国家的友方单位，则己方总部会受到士气伤害
- **收缴**：主动消灭敌方单位后，将其变为1攻1防的复制品加入手牌

## 使用方法

1. 打开 `index.html` 进入主菜单
2. 点击"开始游戏"进入游戏
3. 拖拽手牌到支援阵线或前线
4. 点击单位进行移动或攻击

## 卡牌数据结构

卡牌数据应包含以下字段：
```javascript
{
    id: 1,                    // 卡牌ID
    name: '卡牌名称',         // 卡牌名称
    cost: 2,                  // 费用（指挥点）
    attack: 2,                // 攻击力
    health: 3,                // 生命值
    type: 'infantry',         // 类型：infantry/tank/artillery/fighter/bomber
    nation: 'GER',            // 国家：GER/USSR/US/UK/JAP
    keywords: ['blitz'],      // 关键词数组
    description: '描述',      // 卡牌描述
    rarity: 'common'          // 稀有度：common/limited/special/elite
}
```
