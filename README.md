# 🎓 爱哲安民未来学校校园模拟器

一款专为新生设计的2D俯视角校园生活模拟游戏，通过游戏化的方式帮助新生快速熟悉校园环境、了解学校文化和融入校园生活。

---

## 🌐 网站模式（新功能！）

现在游戏已经可以作为完整的网站运行了！

### 快速启动网站

**Windows系统：**
1. 双击运行 `启动网站服务器.bat`
2. 浏览器会自动打开网站首页
3. 如果没有自动打开，访问：`http://localhost:8080/site.html`

**Mac/Linux系统：**
```bash
python3 启动网站服务器.py
```

然后访问：`http://localhost:8080/site.html`

### 网站功能

- ✅ **网站首页** - 精美的入口页面，展示所有功能
- ✅ **统一导航** - 所有功能页面都可以通过网站访问
- ✅ **响应式设计** - 支持桌面、平板、手机访问
- ✅ **完整功能** - 游戏的所有功能都可以通过网站使用

📖 **详细说明：** 查看 `网站使用指南.md`

## 🚀 快速开始

### 方式一：网站模式（推荐）

1. 启动网站服务器（见上方）
2. 访问网站首页
3. 点击"立即开始"进入登录页面

1. 双击打开 `login.html`（登录页面）

## 🎮 游戏功能

### 核心功能

- **2D校园探索** - 使用WASD键自由探索校园
- **AI智能对话** - 与NPC进行智能对话，了解校园
- **积分系统** - 完成任务获得积分奖励
- **虚拟形象** - 个性化装扮你的角色
- **校园论坛** - 发帖交流，分享学习心得
- **小游戏中心** - 6种休闲益智游戏
- **排行榜系统** - 展示你的成长轨迹
- **好友系统** - 添加好友，建立社交

### 游戏流程

```
登录页面 → 主菜单 → 开始游戏
    ↓
校园探索 → 完成任务 → 获得积分
    ↓
积分商城 → 兑换奖励 → 装扮形象
```

## ⚠️ 注意事项

1. **数据存储** - 游戏数据存储在浏览器LocalStorage中，清除缓存会丢失数据
2. **浏览器要求** - 推荐使用Chrome/Edge等现代浏览器
3. **后端服务器** - 论坛、好友等在线功能需要启动后端服务器

## 🐛 常见问题

### 网站无法访问？

1. 确保服务器已启动
2. 检查端口是否被占用
3. 查看 `网站使用指南.md` 中的"常见问题"章节

### 功能无法使用？

1. 检查浏览器控制台是否有错误
2. 确保后端服务器已启动（在线功能需要）
3. 查看相关文档说明
