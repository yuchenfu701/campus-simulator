# 3D虚拟教室 - 第一人称漫游

这是一个基于Three.js的3D虚拟教室场景，玩家可以在真实的教室环境中自由漫游。

## 场景特点

- 真实还原的中国学校教室场景
- 完整的教室设施（课桌椅、黑板、窗户、投影仪等）
- 真实的3D角色模型，具有完整的走动动画系统
- 第一人称和第三人称视角切换
- 基于Three.js官方示例的自然走动动画
- 完整的物理系统（重力、跳跃、碰撞检测）
- 丰富的角色动作（行走、奔跑、跳跃、滑铲、下蹲、飞行）
- 强大的控制台系统，支持场景内命令输入

## 走动动画系统

游戏采用了基于Three.js官方示例的先进走动动画系统：

### 第一人称视角动画
- **手臂摆动**: 左右手臂自然交替摆动，模拟真实行走
- **腿部动画**: 腿部前后摆动，配合手臂形成协调的步行动作
- **头部摆动**: 轻微的上下和左右摆动，增强沉浸感
- **呼吸效果**: 静止时的自然呼吸动画

### 第三人称视角动画
- **身体摇摆**: 行走时身体的自然左右摇摆
- **前后倾斜**: 根据步行节奏的身体前后倾斜
- **垂直摆动**: 模拟真实行走的上下起伏

### 动画特效
- **脚步声**: 根据步行节奏播放脚步声效果
- **地面粒子**: 行走时产生的地面尘土粒子效果
- **平滑过渡**: 所有动画状态间的平滑过渡

## 操作方法

- **W键**: 向前移动
- **S键**: 向后移动
- **A键**: 向左移动
- **D键**: 向右移动
- **鼠标点击**: 锁定鼠标指针（按ESC解锁）
- **鼠标拖拽**: 控制视角方向
- **空格键**: 跳跃
- **Q键**: 滑铲
- **Shift键**: 下蹲
- **X键**: 切换第一/第三人称视角
- **波浪号(~)或反引号(`)**: 打开/关闭控制台

## 控制台系统

游戏内置了一个强大的控制台系统，可通过按`~`或`` ` ``键打开。支持以下命令：

- **help**: 显示帮助信息
- **clear**: 清空控制台
- **teleport/tp**: 传送到指定坐标，用法: `teleport x y z`
- **fly**: 切换飞行模式
- **speed**: 设置移动速度，用法: `speed [数值]`
- **noclip**: 开启/关闭穿墙模式
- **god**: 开启/关闭无敌模式
- **camera**: 切换摄像机视角，用法: `camera [first/third]`
- **respawn**: 重生到起始位置
- **jump**: 设置跳跃高度，用法: `jump [数值]`
- **day/night**: 设置为白天/夜晚模式
- **size**: 设置玩家大小，用法: `size [数值]`
- **fps**: 显示当前帧率
- **version**: 显示游戏版本信息
- **restart**: 重启游戏
- **gravity**: 设置重力，用法: `gravity [数值]`
- **quit**: 退出游戏

控制台支持命令历史记录（上下箭头键浏览）和命令自动补全（Tab键）。

## 场景内容

一个真实还原的中国学校教室，包含：

1. **教学区**：三联绿色黑板、木质讲台
2. **学生区**：5行5列共25套蓝灰色课桌椅
3. **采光区**：左侧墙面3扇大窗户
4. **教学设备**：天花板投影仪、日光灯管
5. **装饰区**：后墙公告栏/黑板报

## 技术实现

游戏使用了以下技术和库：

- **Three.js (r128)**: 用于3D渲染
- **GLTFLoader**: 加载3D角色模型
- **OrbitControls**: 实现相机控制
- **Canvas API**: 程序化生成迷宫纹理
- **自定义动画系统**: 基于Three.js官方示例的走动动画
- **物理引擎**: 自制的重力和碰撞检测系统

## 场景设计

- 教室由7×7网格组成，每个单元格大小为4个单位
- 纯白色墙壁，浅灰色地面，符合真实教室风格
- 角色碰撞检测使用圆柱体碰撞模型
- 支持多种移动模式（正常行走、飞行、穿墙等）
- 真实的教室家具和设备布局

## 安装和运行
