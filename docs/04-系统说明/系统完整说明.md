## 📱 页面功能详解

### 1. 启动页 (start.html)
**功能**：游戏启动动画，自动跳转到登录页
- 显示游戏Logo
- 加载进度条动画
- 2秒后自动跳转

### 2. 登录界面 (login.html) 👤
**功能**：用户登录系统

#### 登录方式
1. **完整登录**
   - 输入昵称（必填，最多12个字）
   - 输入手机号码（必填，11位数字）
   - 上传头像（可选）

2. **快速登录**
   - 👨‍🎓 学生登录（默认学生账号）
   - 👨‍🏫 教师登录（默认教师账号）
   - 👤 游客登录（默认游客账号）

#### 特性
- 动态粒子背景效果
- 渐变色彩动画
- 自动登录（24小时内）
- 头像上传预览

## 🎨 自定义建议

### 修改商品
编辑 `js/points-system.js`：
```javascript
rewardItems: [
    { id: 'new_item', name: '新商品', price: 30, ... }
]
```

### 修改积分规则
编辑 `js/points-system.js`：
```javascript
pointsRules: {
    '新行为': 15
}
```

### 修改充值档位
编辑 `recharge.html`：
```html
<div class="recharge-option" onclick="selectRecharge(100, 10)">
```

### 修改主题色彩
编辑各HTML文件的CSS：
```css
background: linear-gradient(135deg, #新颜色1, #新颜色2);
```

## 🔒 注意事项

1. **数据安全**
   - 定期导出LocalStorage数据
   - 不要清除浏览器缓存
   - 建议定期备份

2. **浏览器要求**
   - Chrome/Edge（推荐）
   - Firefox
   - Safari
   - 不支持IE

3. **性能优化**
   - 避免同时打开过多页面
   - 定期清理历史记录
   - 建议使用现代浏览器

4. **Token管理**
   - AI对话需要有效的Coze Token
   - Token有时效性，需定期更新
   - 在对应HTML文件中修改

## 🆘 常见问题

### Q: 登录后数据丢失？
A: 可能清除了浏览器缓存，建议不要清除LocalStorage。

### Q: AI对话无法使用？
A: 检查Coze Token是否有效，网络是否正常。

### Q: 充值后积分未到账？
A: 刷新页面，或检查浏览器控制台错误。

### Q: 虚拟形象无法装备？
A: 确保已在商城兑换该物品，查看背包是否有。

### Q: 如何添加新学生？
A: 编辑 `js/points-system.js`，在 `students` 对象中添加。
