# 校园积乐园 - 积分系统使用说明 🎓

## 📖 系统简介

校园积乐园是一个游戏化的学习激励系统，通过积分机制鼓励学生在学习中的优秀表现，学生可以用积分兑换虚拟道具和实物奖励。

---

## 🎯 核心功能

### 1. 积分管理 💎
- **积分获取**：老师根据学生表现发放积分
- **积分记录**：完整的积分获得和消费历史
- **积分规则**：
  - 听写全对：10分
  - 作业优秀：5分
  - 课堂发言：2分
  - 帮助同学：3分
  - 考试进步：20分
  - 完美出勤：5分

### 2. 积分商城 🏪
- **虚拟商品**：发型、眼镜、帽子、称号、背景等
- **实物奖励**：薯片、糖果、文具、手办等
- **即时兑换**：虚拟物品即时到账，实物需找老师领取

### 3. 虚拟形象 👤
- **个性装扮**：自定义发型、眼镜、帽子
- **形象展示**：在个人页面展示独特形象
- **装备管理**：随时更换已拥有的装扮

### 4. 任务系统 📋
- **每日任务**：
  - 每日登录（5分）
  - 完成作业（10分）
  - 阅读30分钟（8分）
- **成就系统**：解锁成就获得额外奖励

### 5. 排行榜 🏆
- **积分榜单**：展示前50名学生
- **班级排行**：与同学良性竞争
- **高亮显示**：自己的排名醒目标记

## 🚀 快速开始

### 学生端使用

#### 1. 打开积分商城
在游戏主界面，点击底部的 **"💎 积分商城"** 按钮

#### 2. 浏览商品
- 全部商品
- 虚拟商品（发型、眼镜、帽子等）
- 实物奖励（零食、文具、玩具等）

#### 3. 兑换物品
1. 点击想要的商品
2. 确认积分足够
3. 点击"确认兑换"
4. 虚拟物品立即到账，实物等待老师发放

#### 4. 装扮形象
1. 点击底部 **"👤 形象"** 标签
2. 选择发型、眼镜、帽子
3. 点击装备，立即生效

#### 5. 查看排行
1. 点击 **"🏆 排行榜"** 标签
2. 查看自己的排名
3. 看看谁是积分之王

#### 6. 完成任务
1. 点击 **"📋 任务"** 标签
2. 查看每日任务
3. 完成任务获得积分

---

## 👨‍🏫 教师端使用

### 1. 打开管理后台
点击游戏主界面的 **"👨‍🏫 教师后台"** 按钮

### 2. 发放积分
#### 快速发放：
1. 选择学生
2. 选择行为类型（自动填充积分）
3. 点击"确认发放"

#### 自定义发放：
1. 选择学生
2. 选择"自定义"行为
3. 手动输入积分和备注
4. 点击"确认发放"

### 3. 查看学生情况
1. 点击"学生管理"卡片
2. 查看所有学生的积分情况
3. 点击"查看记录"查看详细历史

### 4. 兑换审核（开发中）
审核学生的实物奖励兑换申请

### 5. 数据统计（开发中）
查看班级整体积分数据和趋势分析

## 💡 使用技巧

### 学生篇
1. **每日登录**：记得每天登录完成任务，积少成多
2. **目标明确**：先看好想要的物品，然后努力获取积分
3. **良性竞争**：关注排行榜，与同学一起进步
4. **及时装扮**：兑换新装扮后记得去装备展示

### 教师篇
1. **及时奖励**：学生表现好时立即发放积分，强化正向行为
2. **公平公正**：按照统一标准发放积分
3. **定期查看**：查看学生积分情况，了解学习状态
4. **鼓励互助**：对帮助同学的行为给予积分奖励

## 🔧 技术说明

### 技术栈
- **前端**：纯 HTML + CSS + JavaScript
- **存储**：LocalStorage（浏览器本地存储）
- **无需服务器**：所有数据本地保存

### 浏览器兼容
- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ⚠️ IE不支持

### 数据管理
- **备份**：可导出 localStorage 数据备份
- **清除**：清除浏览器缓存会丢失数据，请谨慎
- **同步**：目前不支持多设备同步

## 🔒 注意事项

1. **数据安全**
   - 定期备份数据（导出 localStorage）
   - 不要清除浏览器缓存

2. **积分发放**
   - 建立明确的发放标准
   - 保持公平公正
   - 及时记录

3. **物品兑换**
   - 实物奖励需要实际库存
   - 及时发放已兑换的实物
   - 定期更新商品列表

4. **系统使用**
   - 使用现代浏览器访问
   - 保持网络连接（虽然是本地系统）
   - 遇到问题刷新页面

## 🆘 常见问题

### Q: 积分会过期吗？
A: 不会，积分永久有效。

### Q: 可以赠送积分给同学吗？
A: 目前不支持，只能由老师发放。

### Q: 兑换的实物多久能拿到？
A: 需要找老师领取，具体时间由老师安排。

### Q: 虚拟装扮可以退换吗？
A: 不支持退换，但可以随时更换装备。

### Q: 如何查看积分历史？
A: 在积分商城页面顶部可以看到当前积分，教师后台可查看详细记录。

### Q: 数据丢失了怎么办？
A: 如果清除了浏览器缓存，数据会丢失且无法恢复。建议定期备份。
