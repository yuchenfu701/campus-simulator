# 🔧 商城购买问题修复说明

## 📋 问题描述

**问题**：购买一个物品后，只能兑换任务，无法购买其他商品。

**症状**：
- 购买商品后，商品卡片无法点击
- 只能点击任务卡片
- 模态框可能没有完全关闭

---

## 🔍 问题原因分析

### 1. **模态框关闭不彻底**
- 模态框关闭时，虽然移除了 `show` 类，但可能 `display` 属性没有立即生效
- 遮罩层可能仍然覆盖在页面上，阻止了商品卡片的点击

### 2. **状态重置不完整**
- `selectedItem` 可能在某些情况下没有正确清空
- 确认按钮状态可能没有正确重置

### 3. **事件处理问题**
- 没有点击遮罩层关闭的功能
- 没有ESC键关闭的功能

---

## ✅ 修复方案

### 1. **优化模态框关闭逻辑**

**修改前**：
```javascript
function closeModal() {
    modal.classList.remove('show');
    modal.style.display = 'none';
    setTimeout(() => {
        modal.style.display = '';
    }, 500); // 延迟恢复display，可能导致问题
    selectedItem = null;
}
```

**修改后**：
```javascript
function closeModal() {
    const modal = document.getElementById('exchangeModal');
    if (modal) {
        modal.classList.remove('show');
        modal.style.display = 'none'; // 立即隐藏，不延迟恢复
    }
    selectedItem = null;
    // 重置确认按钮状态
    const confirmBtn = document.querySelector('#exchangeModal .btn-primary');
    if (confirmBtn) {
        confirmBtn.style.display = 'block';
    }
}
```

### 2. **优化购买确认流程**

**修改前**：
```javascript
function confirmExchange() {
    const result = pointsSystem.exchangeReward(userId, selectedItem.id);
    closeModal(); // 先关闭
    setTimeout(() => {
        updateUI();
        renderItems();
        showToast(...);
    }, 100); // 延迟更新
}
```

**修改后**：
```javascript
function confirmExchange() {
    // 保存商品信息
    const itemId = selectedItem.id;
    
    const result = pointsSystem.exchangeReward(userId, itemId);
    
    // 立即关闭模态框
    const modal = document.getElementById('exchangeModal');
    if (modal) {
        modal.classList.remove('show');
        modal.style.display = 'none';
    }
    
    // 立即重置状态
    selectedItem = null;
    
    // 立即更新UI和商品列表
    updateUI();
    renderItems();
    
    // 显示提示
    showToast(...);
}
```

### 3. **优化显示模态框逻辑**

**修改后**：
```javascript
function showExchangeModal(item) {
    // 确保之前没有选中的商品
    selectedItem = null;
    
    // 重置模态框状态
    const modal = document.getElementById('exchangeModal');
    if (modal) {
        modal.style.display = 'flex'; // 确保显示
    }
    
    selectedItem = item;
    // ... 设置模态框内容 ...
    
    // 显示模态框
    document.getElementById('exchangeModal').classList.add('show');
}
```

### 4. **添加点击遮罩层关闭功能**

**HTML修改**：
```html
<!-- 添加点击事件 -->
<div class="modal" id="exchangeModal" onclick="handleModalClick(event)">
    <div class="modal-content" onclick="event.stopPropagation()">
        <!-- 内容 -->
    </div>
</div>
```

**JavaScript添加**：
```javascript
function handleModalClick(event) {
    // 如果点击的是遮罩层（不是modal-content），则关闭模态框
    if (event.target.id === 'exchangeModal') {
        closeModal();
    }
}
```

### 5. **添加ESC键关闭功能**

**添加**：
```javascript
document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') {
        const exchangeModal = document.getElementById('exchangeModal');
        if (exchangeModal && exchangeModal.classList.contains('show')) {
            closeModal();
        }
    }
});
```

### 6. **确保页面加载时模态框隐藏**

**添加**：
```javascript
window.addEventListener('DOMContentLoaded', () => {
    // 确保模态框初始状态为隐藏
    const exchangeModal = document.getElementById('exchangeModal');
    if (exchangeModal) {
        exchangeModal.style.display = 'none';
        exchangeModal.classList.remove('show');
    }
    // ...
});
```
