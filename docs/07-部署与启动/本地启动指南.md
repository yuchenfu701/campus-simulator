# 🚀 本地启动指南

## 一、启动前端（无需后端）

直接用浏览器打开 `index.html` 即可，或使用本地服务器：

```bash
# Python 3
python -m http.server 8080

# 然后访问
http://localhost:8080/
```

或双击运行 `启动网站服务器.bat`，浏览器会自动打开。

---

## 二、启动后端服务器（论坛/好友功能需要）

### 前置要求

- **Node.js**：https://nodejs.org/
- **MongoDB**：本地安装，或使用MongoDB Atlas云服务

### 启动步骤

```bash
# 1. 进入server目录
cd server

# 2. 安装依赖（首次运行）
npm install

# 3. 启动服务器
npm start
```

或双击运行 `启动服务器.bat`。

### 验证服务器正常运行

访问：`http://localhost:3000/api/health`

返回 `{"success": true}` 即正常。

## 四、常见问题

**端口被占用：** 修改脚本中的端口号，或关闭占用端口的程序。

**MongoDB连接失败：** 确保MongoDB服务正在运行，连接字符串：`mongodb://localhost:27017`

**依赖安装失败：** 使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```
