# 🌐 公网部署指南 - 让所有人通过网址访问你的游戏

本指南将帮助你将游戏部署到互联网，让任何人可以直接通过网址访问，无需下载文件或启动本地服务器。

---

## 🎯 推荐方案对比

| 方案 | 优点 | 缺点 | 适合场景 |
|------|------|------|----------|
| **GitHub Pages** ⭐ | 完全免费、简单、稳定 | 只能部署静态文件 | 纯前端项目 |
| **Vercel** ⭐⭐ | 免费、快速、自动HTTPS | 需要GitHub账号 | 前端+后端API |
| **Netlify** ⭐⭐ | 免费、功能强大、易用 | 需要GitHub账号 | 前端+后端API |
| **Railway** | 支持后端服务器 | 有免费额度限制 | 需要后端功能 |

---

## 🚀 方案一：GitHub Pages（最简单，推荐）

### 优点
- ✅ **完全免费**，无限制
- ✅ **简单易用**，几分钟完成
- ✅ **稳定可靠**，GitHub官方服务
- ✅ **自动HTTPS**，安全访问
- ✅ **全球CDN**，访问速度快

### 部署步骤

#### 第一步：准备GitHub账号

1. **注册GitHub账号**（如果还没有）
   - 访问：https://github.com/
   - 点击 "Sign up" 注册
   - 验证邮箱

#### 第二步：创建GitHub仓库

1. **创建新仓库**
   - 登录GitHub后，点击右上角 "+" → "New repository"
   - Repository name: `campus-simulator`（或你喜欢的名字）
   - Description: "爱哲安民未来学校校园模拟器"
   - 选择 **Public**（公开，GitHub Pages需要）
   - **不要**勾选 "Initialize this repository with a README"
   - 点击 "Create repository"

#### 第三步：上传代码到GitHub

**使用命令行（推荐，已配置好）**

```bash
cd "D:/编程/test"
git add .
git commit -m "update"
git push origin main
```

#### 第四步：启用GitHub Pages

1. 在GitHub仓库页面，点击 "Settings" → 左侧 "Pages"
2. Source: 选择 "Deploy from a branch"
3. Branch: 选择 "main"，Folder: "/ (root)"
4. 点击 "Save"
5. 等待几分钟，访问：`https://yuchenfu701.github.io/campus-simulator/`

## 🚀 方案二：Vercel（推荐，功能强大）

1. 访问：https://vercel.com/ → 用GitHub登录
2. 点击 "Add New..." → "Project" → 选择仓库
3. Framework Preset: "Other"，其他留空，点击 "Deploy"
4. 访问：`https://campus-simulator.vercel.app`

---

## 🚀 方案三：Netlify

1. 访问：https://www.netlify.com/ → 用GitHub登录
2. "Add new site" → "Import an existing project" → 选择仓库
3. Build command 留空，Publish directory: `.`
4. 点击 "Deploy site"

## 🐛 常见问题

### Q1: 部署后显示404？
等待几分钟，或检查GitHub Pages设置是否正确。

### Q2: 网站更新后没有变化？
清除浏览器缓存（Ctrl+F5），或等待几分钟CDN更新。

### Q3: 图片或资源无法加载？
检查文件路径是否正确（使用相对路径），确保所有文件已上传。
