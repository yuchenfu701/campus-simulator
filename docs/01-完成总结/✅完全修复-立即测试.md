## 💡 为什么会出现这个错误？

### 问题原因
当用户第一次登录时，`login.html` 中的 `initUserPointsData()` 函数创建新用户数据，但只初始化了基本字段：
```javascript
pointsData.students[userId] = {
    id: userId,
    name: nickname,
    points: 100,
    totalEarned: 100,
    inventory: [],
    avatar: {...},
    achievements: [],
    tasks: {}
};
```

**注意：缺少了 `pointsHistory` 和 `exchangeHistory` 字段！**

当任务系统调用 `addPoints()` 时：
```javascript
student.pointsHistory.push({...});  // ❌ pointsHistory is undefined
```

### 解决方案
在所有使用数组的地方，先检查数组是否存在：
```javascript
if (!student.pointsHistory) {
    student.pointsHistory = [];
}
student.pointsHistory.push({...});  // ✅ 现在安全了
```
