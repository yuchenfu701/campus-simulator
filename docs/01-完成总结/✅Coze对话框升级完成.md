## 🔧 配置说明

### Coze SDK 配置项

#### 必需配置
```javascript
config: {
    bot_id: '7584781738845945875',  // Bot ID（必需）
}
```

#### 身份认证
```javascript
auth: {
    type: 'token',           // 认证类型
    token: API_TOKEN,        // 访问令牌
    onRefreshToken: () => API_TOKEN  // 刷新令牌回调
}
```

#### UI配置
```javascript
ui: {
    base: {
        icon: '图标URL',     // 自定义图标
        layout: 'pc',        // 布局模式
        zIndex: 9999,        // 层级
    },
    asstBtn: {
        isNeed: false,       // 是否显示助手按钮
    }
}
```

#### 组件属性

componentProps: {

}

## 📞 问题排查

### 常见问题

#### Q1: 点击后没反应？
**A**: 检查以下几点：
```
1. 打开控制台（F12）查看是否有错误
2. 确认看到 "✅ Coze SDK 初始化成功"
3. 刷新页面（Ctrl + Shift + R）
4. 检查网络连接
```

#### Q2: 提示初始化失败？
**A**: 可能的原因：
```
1. Token失效 - 更新Token
2. Bot ID错误 - 检查Bot ID
3. 网络问题 - 检查网络连接
4. SDK加载失败 - 刷新页面
```

#### Q3: AI回复很慢？
**A**: 这是正常现象：
```
1. AI需要时间思考和生成回复
2. 网络延迟可能影响速度
3. 复杂问题需要更长时间
4. 可以在等待时看到加载状态
```

#### Q4: ESC键不起作用？
**A**: 尝试以下方法：
```
1. 点击页面空白处获取焦点
2. 使用窗口的关闭按钮
3. 刷新页面重试
```
