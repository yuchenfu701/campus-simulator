# 🧪 教师办公室 - 快速测试指南

## 🎯 测试目标
验证新添加的教师办公室是否正常工作

---

## 📝 测试步骤

### 第一步：进入游戏
```
1. 打开浏览器
2. 访问游戏主页（或打开 start.html）
3. 登录你的账号
```

### 第二步：找到教师办公室
```
1. 进入校园地图
2. 在地图上寻找 "食堂" 和 "G8教学楼"
3. 教师办公室就在它们中间
4. 是一个浅蓝色的建筑
```

**定位提示**：
- 🍜 食堂在左边
- 🏫 教师办公室在中间（浅蓝色）
- 📚 G8教学楼在右边

### 第三步：靠近建筑
```
1. 操作角色走向教师办公室
2. 走到建筑门口附近
```

### 第四步：按E键进入
```
1. 走近后，屏幕应该会显示提示
   "按 E 键进入教师办公室"
2. 按下键盘上的 E 键
3. 应该会进入办公室内部场景
```

### 第五步：查看办公室布局
```
进入后，你应该看到：

左侧：
- 📌 名牌："英语老师 - 李响"
- 📚 书架（装满书）
- 💻 电脑和键盘
- 🪴 桌上有盆栽
- 🪑 办公椅

右侧：
- 📌 名牌："办公位（暂未分配）"
- 同样完整的办公设施
- 🔒 暂无老师
```

### 第六步：与李响老师对话
```
1. 点击左侧李响老师的办公位
   （点击办公桌、名牌或整个区域都可以）
2. 应该弹出对话框
3. 看到欢迎消息：
   "Hi! 我是李响老师。有什么可以帮你的吗？..."
```

### 第七步：测试AI对话
```
1. 在对话框底部的输入框中输入：
   "你好，李老师！"
2. 点击"发送"按钮或按回车
3. 等待几秒
4. 应该收到李响老师的AI回复
```

**测试对话示例**：
- "你好，李老师！"
- "怎样才能学好英语？"
- "有什么学习建议吗？"
- "如何提高口语？"

### 第八步：测试右侧办公位
```
1. 关闭与李响老师的对话框
2. 点击右侧的办公位
3. 应该弹出提示：
   "这个办公位暂时还没有老师入驻哦！"
```

### 第九步：退出办公室
```
方式1：
- 点击右上角的 "← 返回校园" 按钮

方式2：
- 按下键盘上的 ESC 键

结果：
- 应该返回到校园地图
- 你的角色在教师办公室门口
```

## 🐛 可能的问题和解决方案

### 问题1：找不到教师办公室
**症状**：在地图上看不到教师办公室建筑

**解决方案**：
```
1. 刷新页面（Ctrl + Shift + R 强制刷新）
2. 清除浏览器缓存
3. 检查 js/map.js 文件是否正确更新
```

### 问题2：无法按E键进入
**症状**：走到门口没有提示，或按E键无反应

**解决方案**：
```
1. 确保走得足够近（要进入建筑的交互范围）
2. 检查游戏引擎是否支持E键交互
3. 查看浏览器控制台是否有错误信息
```

### 问题3：进入后页面空白
**症状**：按E键后屏幕变白或显示不正常

**解决方案**：
```
1. 检查 office-teachers.html 文件是否存在
2. 查看浏览器控制台的错误信息
3. 确认文件路径是否正确
```

### 问题4：AI对话无响应
**症状**：发送消息后没有回复，或一直显示"正在思考..."

**解决方案**：
```
1. 检查网络连接是否正常
2. 确认API密钥是否有效
3. 等待更长时间（有时API响应较慢）
4. 查看浏览器控制台的错误信息
```

### 问题5：点击办公位无反应
**症状**：点击办公桌或名牌没有任何反应

**解决方案**：
```
1. 刷新页面重试
2. 检查浏览器控制台是否有JavaScript错误
3. 确认点击的是可交互区域（整个办公位都可点击）
```
