## 🎯 任务规则

### 与其他任务保持一致 ✅

所有教师任务都遵循相同的规则：

1. **激活码验证**
   - 玩家需要从对应老师处获得激活码
   - 输入正确的激活码才能完成任务

2. **积分奖励**
   - 完成任务后获得 **100积分**
   - 积分立即到账

3. **任务状态**
   - 完成任务后，任务从列表中消失
   - 不能重复领取奖励

4. **提示信息**
   - 输入激活码后显示Toast提示："您已获得 100 积分"
   - 激活码错误时提示："❌ 激活码错误，请检查后重试"

## 🔧 技术实现

### 代码位置

**文件**: `points-mall.html`

**位置**: JavaScript任务定义数组（约第532行）

### 代码结构

```javascript
const teacherTasks = [
    // ... 原有任务 ...
    {
        id: 'task_english_teacher_katie',
        name: '英语老师凯蒂任务',
        icon: '👩‍🏫',
        description: '找英语老师凯蒂对话，完成任务领取激活码',
        reward: 100,
        activationCode: '576890138',
        teacher: '英语老师凯蒂',
        category: 'task'
    },
    {
        id: 'task_english_teacher_lixiang',
        name: '英语老师李响任务',
        icon: '👨‍🏫',
        description: '找英语老师李响对话，完成任务领取激活码',
        reward: 100,
        activationCode: '15267683145',
        teacher: '英语老师李响',
        category: 'task'
    }
];
```

### 字段说明

- **id**: 任务的唯一标识符
- **name**: 任务显示名称
- **icon**: 任务图标（emoji）
- **description**: 任务描述文本
- **reward**: 奖励积分数量
- **activationCode**: 激活码（字符串）
- **teacher**: 对应的老师名称
- **category**: 分类（固定为'task'）

## 🚀 使用流程

### 玩家操作步骤

#### 步骤1：进入积分商城
```
从主菜单进入"积分商城"
或直接访问 points-mall.html
```

#### 步骤2：查看任务列表
```
点击顶部导航栏的"📋 任务"标签
或查看"全部"分类中的任务卡片
```

#### 步骤3：选择英语老师任务
```
点击"英语老师凯蒂任务"或"英语老师李响任务"卡片
```

#### 步骤4：获取激活码
```
前往对应老师处对话：
- 英语老师凯蒂：G8教室左侧（office.html）
- 英语老师李响：教师办公室左侧（office-teachers.html）
```

#### 步骤5：输入激活码
```
在弹窗中输入从老师处获得的激活码：
- 凯蒂老师：576890138
- 李响老师：15267683145
```

#### 步骤6：领取奖励
```
✅ 激活码正确 → 获得100积分 → 任务消失
❌ 激活码错误 → 提示错误 → 重新输入
```

### 对于开发者

#### 如何添加更多任务？

1. **在teacherTasks数组中添加新任务对象**
```javascript
{
    id: 'task_unique_id',
    name: '任务名称',
    icon: '🎯',
    description: '任务描述',
    reward: 100,
    activationCode: '激活码',
    teacher: '老师名称',
    category: 'task'
}
```

2. **确保字段完整**
- id必须唯一
- activationCode为字符串类型
- reward为数字类型

3. **测试新任务**
- 检查任务是否正确显示
- 验证激活码是否正确
