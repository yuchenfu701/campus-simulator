```javascript

const API_TOKEN = 'pat_yMg90FTTloleoHgkyZOO4ix71zoYR8i3sz90YgOTJtoacKJBWT5AzMt4Hi2paxa2';
```

### 2. **初始化Coze SDK弹窗客户端** ✅
```javascript
cozeWebSDK1 = new CozeWebSDK.WebChatClient({
    config: {
        bot_id: BOT_ID,
    },
    componentProps: {
        title: '李响老师 - 英语学习助手',
    },
    ui: {
        base: {
            icon: 'https://lf-coze-web-cdn.coze.cn/obj/coze-web-cn/obric/coze/favicon.1970.png',
            layout: 'pc',
            zIndex: 9999,
        },
        asstBtn: {
            isNeed: false,  // 不显示助手按钮
        },
    },
    auth: {
        type: 'token',
        token: API_TOKEN,
        onRefreshToken: function () {
            return API_TOKEN;
        }
    },
});
```

### 3. **修改对话触发函数** ✅
```javascript
// 新的实现
function interactWithTeacher(teacherName) {
    console.log('与老师对话:', teacherName);
    
    if (cozeWebSDK1) {
        // 显示 Coze 官方对话框
        cozeWebSDK1.showChatBot();
        console.log('💬 已打开与李响老师的对话窗口');
    } else {
        console.error('❌ Coze SDK 未初始化');
        alert('对话系统初始化中，请稍后再试！');
    }
}
```

### 4. **更新ESC键行为** ✅
```javascript
// ESC键现在会隐藏Coze对话框
document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape') {
        if (cozeWebSDK1) {
            try {
                cozeWebSDK1.hideChatBot();
                console.log('💬 已关闭对话窗口');
            } catch (error) {
                exitRoom();
            }
        } else {
            exitRoom();
        }
    }
});
```

## 🔧 技术细节

### SDK配置说明

#### 1. **bot_id**
```javascript
bot_id: '7584781738845945875'
```
- 李响老师的Bot ID
- 唯一标识这个AI助手

#### 2. **title**
```javascript
title: '李响老师 - 英语学习助手'
```
- 对话窗口的标题
- 显示在窗口顶部

#### 3. **layout**
```javascript
layout: 'pc'
```
- 布局模式：PC端
- 也可以设置为 'mobile'

#### 4. **zIndex**
```javascript
zIndex: 9999
```
- 窗口的层级
- 确保对话框在最上层

#### 5. **asstBtn**
```javascript
asstBtn: {
    isNeed: false
}
```
- 不显示助手按钮
- 因为我们用点击办公位来触发

#### 6. **auth**
```javascript
auth: {
    type: 'token',
    token: API_TOKEN,
    onRefreshToken: function () {
        return API_TOKEN;
    }
}
```
- 身份验证方式：Token
- 提供刷新Token的回调函数

## 📞 问题排查

### 问题1：点击后没有弹窗
**可能原因**：
- SDK未正确初始化
- Token失效
- Bot ID错误

**解决方案**：
```
1. 查看控制台是否有 "✅ Coze SDK 初始化成功"
2. 检查Token是否正确
3. 确认Bot ID是否正确
```

### 问题2：弹窗显示错误
**可能原因**：
- 网络连接问题
- API限制
- Bot配置问题

**解决方案**：
```
1. 检查网络连接
2. 查看控制台错误信息
3. 联系Coze技术支持
```

### 问题3：ESC键无效
**可能原因**：
- 事件监听未正确绑定
- 页面焦点问题

**解决方案**：
```
1. 点击页面空白处获取焦点
2. 刷新页面重试
3. 使用对话框的关闭按钮
```
