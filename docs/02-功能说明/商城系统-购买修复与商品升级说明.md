## ✅ 问题修复

### 购买后无法再次购买的问题 🔧

**问题原因**：
- 当玩家积分不足时，确认按钮会被隐藏
- 关闭模态框后，按钮状态没有重置
- 导致下次打开时按钮仍然隐藏，无法购买

**解决方案**：
```javascript
// 在closeModal函数中添加按钮重置逻辑
function closeModal() {
    // ... 关闭模态框代码 ...
    
    // 确保确认按钮显示（防止下次打开时按钮被隐藏）
    const confirmBtn = document.querySelector('#exchangeModal .btn-primary');
    if (confirmBtn) {
        confirmBtn.style.display = 'block';
    }
}
```

**修复效果**：
- ✅ 每次关闭模态框后，确认按钮状态都会重置
- ✅ 无论之前是否积分不足，下次打开都能正常购买
- ✅ 购买流程更加流畅稳定
