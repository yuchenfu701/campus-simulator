# 🎮 首次启动跳转逻辑说明

## 📋 更新概述

修改了游戏启动逻辑，使第一次启动时先跳转到介绍页面，第二次及以后直接进入游戏。

---

## ✅ 功能说明

### 启动流程

#### 第一次启动 🆕
```
点击"开始游戏" → 跳转到"梦核校园最新力作(2).html" → 播放介绍 → 跳转到index.html
```

#### 第二次及以后 🔄
```
点击"开始游戏" → 直接跳转到index.html
```

---

## 🔧 实现逻辑

### 代码实现

**修改位置**：`main-menu.html` - `startGame()` 函数

**代码**：
```javascript
function startGame() {
    // 检查是否是第一次启动游戏
    const hasStartedBefore = localStorage.getItem('game_started');
    
    if (!hasStartedBefore) {
        // 第一次启动：跳转到介绍页面
        console.log('🎮 首次启动游戏，跳转到介绍页面');
        localStorage.setItem('game_started', 'true');
        window.location.href = '梦核校园最新力作(2).html';
    } else {
        // 第二次及以后：直接跳转到游戏
        console.log('🎮 继续游戏，跳转到主游戏');
        window.location.href = 'index.html';
    }
}
```

### 工作原理

1. **检查标记**
   - 从localStorage读取 `game_started` 标记
   - 如果不存在，说明是第一次启动

2. **第一次启动**
   - 跳转到"梦核校园最新力作(2).html"
   - 设置 `game_started = 'true'` 标记
   - 介绍页面播放完后会自动跳转到index.html

3. **第二次及以后**
   - 检测到标记存在
   - 直接跳转到index.html
   - 跳过介绍页面
