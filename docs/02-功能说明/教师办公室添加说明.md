# 🏫 教师办公室添加说明

## 📍 位置

教师办公室位于**食堂**和**G8教学楼**之间的空地上。

### 地图坐标
- **X坐标**: 240
- **Y坐标**: 720
- **宽度**: 140
- **高度**: 130
- **背景色**: 浅蓝色 (#B8E6F9)

---

## 🎯 功能特点

### 1. **建筑外观**
- 位于食堂（x: 80）和G8（x: 400）之间
- 浅蓝色建筑，现代化设计
- 带有窗户和细节装饰
- 底部有入口门

### 2. **进入方式**
✅ **按 E 键进入**
- 玩家走到教师办公室附近
- 屏幕会显示提示：`按 E 键进入`
- 按下 E 键即可进入办公室内部

### 3. **内部布局**

#### 左侧办公位 - 英语老师李响
- **名牌**: "英语老师 - 李响"
- **配置**:
  - 木质办公桌
  - 电脑显示器 + 键盘
  - 书架（4层，装满书籍）
  - 盆栽绿植
  - 办公椅
- **交互功能**: 💬 点击可与李响老师对话
- **AI对话**: 集成了Coze AI，可以进行真实对话

#### 右侧办公位 - 待分配
- **名牌**: "办公位（暂未分配）"
- **配置**:
  - 同样完整的办公设施
  - 暂无老师入驻
- **交互提示**: 🔒 暂无老师在此

### 基本信息
- **姓名**: 李响

## 🎮 使用方法

### 步骤1：走到教师办公室
```
从食堂或G8教学楼方向
走到中间的浅蓝色建筑前
```

### 步骤2：按E键进入
```
看到提示后，按下 E 键
即可进入办公室内部
```

### 步骤3：与老师交互
```
方式1：点击左侧李响老师的办公位
方式2：直接点击办公桌或名牌
```

### 步骤4：进行对话
```
- 在对话框中输入你的问题
- 按回车或点击"发送"按钮
- 李响老师会用AI回复你
```

### 步骤5：退出
```
方式1：点击右上角"返回校园"按钮
方式2：按 ESC 键退出
```

## 🎯 技术要点

### 1. **地图集成**
- 在 `SchoolMap` 类的 `initializeMap()` 方法中添加区域
- 设置正确的坐标、尺寸和属性
- 添加 `specialPage` 属性指向场景HTML

### 2. **NPC系统**
- 在 `initializeNPCs()` 方法中添加李响老师
- 设置老师的属性和对话内容
- 关联到教师办公室区域

### 3. **交互系统**
- 通过 `specialPage` 属性实现按E键进入
- 使用点击事件触发对话
- 集成Coze AI进行智能对话

### 4. **视觉设计**
- 使用CSS渐变创建立体效果
- position: absolute 布局办公位
- hover效果提升交互体验
