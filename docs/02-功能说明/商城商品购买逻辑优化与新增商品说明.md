## ✅ 功能优化

### 1. **虚拟商品购买后消失**

**规则**：
- ✅ **实物商品（real）**：可以重复购买，购买后不消失
- ✅ **虚拟商品（virtual）**：购买一次后从商品列表中消失

**实现逻辑**：
```javascript
// 过滤掉已购买的虚拟商品（实物商品可以重复购买）
const student = pointsSystem.getCurrentStudent();
if (student && student.inventory) {
    filteredItems = filteredItems.filter(item => {
        // 实物商品（real）可以重复购买，不过滤
        if (item.category === 'real') {
            return true;
        }
        // 虚拟商品：如果已经拥有，则从列表中移除
        const hasItem = student.inventory.some(inv => inv.itemId === item.id);
        return !hasItem;
    });
}
```

**效果**：
- 购买虚拟商品后，该商品不再显示在商城中
- 实物商品可以多次购买（受库存限制）
- 提升用户体验，避免重复购买虚拟商品

- 实物商品可以重复购买，但注意库存限制

- 打开"背包"页面

- 可以在"形象"页面装备虚拟商品

### 对于开发者

**代码逻辑**：
- 虚拟商品通过检查 `inventory` 数组过滤
- 实物商品不受此限制
- 商品列表在每次渲染时动态过滤

**扩展商品**：
- 在 `rewardItems` 数组中添加新商品
- 虚拟商品设置 `category: 'virtual'`
- 实物商品设置 `category: 'real'` 和 `stock` 值
