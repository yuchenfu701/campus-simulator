# 🚀 快速部署指南 - 5分钟上线

## 📋 准备工作

1. **GitHub账号**（如果没有，去 https://github.com 注册）
2. **邮箱账号**（用于注册云服务）

---

## 🎯 最简单方案：Railway（推荐）

### 第一步：准备MongoDB Atlas（5分钟）

1. 访问：https://www.mongodb.com/cloud/atlas/register
2. 注册账号（免费）
3. 创建免费集群：
   - 点击 "Create" → "Build a Database"
   - **重要：选择 "Free" 套餐（不要选M10或其他付费套餐！）**
   - Free套餐显示：512 MB存储，RAM共享，VCPU共享
   - 选择地区（选离你最近的）
   - 点击 "Create Deployment"（创建部署）
4. 创建数据库用户：
   - Username: `campus_admin`
   - Password: 设置密码（记住它！）
5. 配置网络访问：
   - 点击 "Network Access" → "Add IP Address"
   - 选择 "Allow Access from Anywhere" (0.0.0.0/0)
6. 获取连接字符串：
   - 点击 "Connect" → "Connect your application"
   - 复制连接字符串
   - 将 `<password>` 替换为你的密码
   - **保存这个字符串！**

---

### 第二步：部署到Railway（5分钟）

1. **注册Railway**
   - 访问：https://railway.app/
   - 点击 "Login" → 使用GitHub登录

2. **创建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 如果没有GitHub仓库：
     - 去GitHub创建新仓库
     - 将 `server` 文件夹的内容上传
     - 在Railway中选择这个仓库

3. **配置环境变量**
   - 在Railway项目页面，点击 "Variables"
   - 添加以下变量：
     ```
     MONGODB_URI = mongodb+srv://campus_admin:你的密码@cluster0.xxxxx.mongodb.net/campus_simulator?retryWrites=true&w=majority
     CORS_ORIGIN = *
     NODE_ENV = production
     ```
   - Railway会自动设置 `PORT`

4. **等待部署**
   - Railway会自动开始部署
   - 等待2-3分钟
   - 看到 "Deployed" 表示成功

5. **获取服务器地址**
   - 点击 "Settings"
   - 找到 "Domains"
   - 复制你的域名（如：`https://your-app.up.railway.app`）
   - **API地址：`https://your-app.up.railway.app/api`**

---

### 第三步：配置前端（1分钟）

1. **打开配置页面**
   - 在浏览器中打开 `server-config.html`

2. **输入服务器地址**
   - 在输入框中输入你的Railway域名（不需要加/api）
   - 例如：`https://your-app.up.railway.app`

3. **测试连接**
   - 点击 "测试连接" 按钮
   - 如果显示 "连接成功"，说明配置正确

4. **保存配置**
   - 点击 "保存配置"
   - 刷新页面

5. **完成！**
   - 现在所有功能都会使用云端服务器
   - 其他人也可以使用你的服务器地址联机

## 🔧 其他部署方案

### Render（免费但会休眠）

1. 访问：https://render.com/
2. 注册账号（GitHub登录）
3. 创建 "New Web Service"
4. 连接GitHub仓库
5. 配置环境变量（同Railway）
6. 部署完成

**注意：** Render免费版会在15分钟无活动后休眠，首次访问需要等待唤醒。

## 🆘 常见问题

### Q: 部署失败怎么办？
A: 检查环境变量是否正确，特别是MONGODB_URI

### Q: 连接数据库失败？
A: 检查MongoDB Atlas的网络访问设置，确保允许所有IP

### Q: 如何更新代码？
A: 推送代码到GitHub，Railway会自动重新部署

### Q: 免费额度够用吗？
A: Railway每月$5免费额度，小型应用足够使用
