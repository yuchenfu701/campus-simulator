# 平等象限 - 性别平等主题桌游

## 项目简介

《平等象限》是一款合作与竞争并存的策略卡牌游戏，主题为性别平等与社会共建。玩家将扮演社会中的不同角色，通过做出决策、应对事件、分配资源，共同将一个初始存在性别偏见的社会，建设成为一个更加平等的"理想国"。

## 游戏信息

- **玩家人数**：2-4人
- **游戏时长**：45-60分钟
- **目标玩家**：14岁及以上
- **适用场景**：家庭教育、学校课堂、团队建设

## 核心玩法

游戏版图分为四个象限，代表社会的四个核心领域：
1. 职场与企业
2. 家庭与社区
3. 教育与文化
4. 政治与法律

玩家需要合作将"社会平等进度条"从10%提升到100%，同时争夺"平等先锋"的个人荣誉。

## 如何开始

1. 用浏览器打开 `index.html` 文件
2. 选择玩家数量（2-4人）
3. 每位玩家选择自己的角色
4. 开始游戏！

## 游戏特色

### 🎨 精美设计
- 采用苹果风格的简洁优雅界面
- 流畅的动画效果和交互反馈
- 响应式设计，适配不同屏幕尺寸

### 🎮 完整游戏体验
- **4个独特角色**，各有特殊技能
- **20+事件卡**，涵盖职场、家庭、教育、政治四大领域
- **12张行动卡**，提供多样化的策略选择
- **6个长期项目**，需要玩家协作完成
- **4个刻板印象**，模拟现实社会中的障碍

### 📚 教育意义
- 让玩家理解系统性偏见的影响
- 培养对性别平等议题的认知
- 体验合作共赢的社会改革过程
- 适合学校、家庭、团队建设使用
