# 《平等象限》游戏改进说明

## 版本 2.0 - 完整版

基于50,000字设计文档的深度改进，游戏现在拥有更复杂、更策略性的机制。

---

## ✅ 已完成的改进

### 1. 增强的角色系统

每个角色现在拥有：

#### 🏛️ 政策制定者
- **被动技能 - 法令颁布**：在政治象限完成项目时，额外获得1影响力和1支持
- **主动技能 - 政治动员**（冷却2回合）：消耗2支持，使所在象限本回合行动消耗减少1影响力
- 初始资源：💪4 💰2 🤝3

#### 💼 企业领袖
- **被动技能 - 商业投资**：在职场象限执行赚取资金行动时，额外获得1资金
- **主动技能 - 经济赋能**（冷却3回合）：支付3资金，为一个项目立即推进2点进度
- 初始资源：💪3 💰5 🤝1

#### 🤝 社区组织者
- **被动技能 - 草根网络**：手牌上限+1。在家庭象限移除刻板印象时，额外+1%进度
- **主动技能 - 团结一心**（冷却2回合）：消耗1支持，使你和同象限的另一名玩家下回合获得额外行动
- 初始资源：💪3 💰1 🤝5
- 特殊：手牌上限6张（其他角色5张）

#### 📚 教育工作者
- **被动技能 - 启迪心智**：在教育象限执行行动后，50%概率抽一张牌
- **主动技能 - 批判性思维**（冷却3回合）：立即免费破解你所在象限的一个刻板印象
- 初始资源：💪4 💰2 🤝3

### 2. 象限本地进度系统

- 每个象限现在有独立的本地进度（0-25%）
- 总社会进度 = 四个象限进度的平均值（每个象限贡献25%）
- 事件和行动可以直接影响特定象限的进度
- 更精准地反映社会各领域的平等状态

### 3. 增强的事件系统

#### 持续效果
- 某些事件有持续回合数（duration），效果会持续多个回合
- 例如："算法性别偏见曝光"会持续2回合影响职场象限

#### 失败条件
- 重要事件如果长期未解决，会触发失败效果
- `failTurns`：未解决的最大回合数
- `failEffect`：失败时的惩罚（通常是进度倒退）
- 例如：事件持续3回合未解决，社会进度-5%

#### 新增事件
新增了6张更有深度的事件卡：
- **算法性别偏见曝光**：关注AI时代的性别偏见
- **职场母职惩罚现象**：揭示女性生育后的职业困境
- **男性育儿假改革**：促进育儿责任分担
- **STEM领域性别差距报告**：关注科技领域性别失衡
- **性别平等教育立法**：推动系统性教育改革

### 4. 改进的项目系统

#### 累计贡献机制
- 项目不再是一次性完成，而是需要累计投入资源
- 记录每个玩家的具体贡献（影响力、资金、支持）
- `requiredTotal`：显示项目需要的具体资源构成
- `progressNeeded`：总进度点数

#### 贡献追踪
```javascript
contributors: {
    player_1: { influence: 3, money: 2, support: 1 },
    player_2: { influence: 1, money: 3, support: 2 }
}
```

#### 奖励分配
- 资源奖励：所有贡献者平分
- 个人分数：按贡献比例分配
- 更公平地反映每个玩家的投入

#### 永久效果
每个项目完成后，会提供永久性的游戏增益：
- "建立普惠托儿所网络"：所有玩家在职场获取资金时+1
- "推动反歧视立法"：移除刻板印象消耗减少1
- "性别平等教育课程"：每回合开始抽1张额外手牌
- "企业平等认证计划"：职场象限本地进度+5%

#### 新增项目
新增了2个项目：
- **反性骚扰举报机制**：建立安全的职场环境
- **STEM女性导师计划**：为女生提供榜样和指导

### 5. 扩展的行动卡库

新增了8张行动卡，提供更多策略选择：
- **跨领域合作**：与其他象限玩家协作
- **数据调研**：为政策提供科学依据
- **公益诉讼**：通过法律途径移除刻板印象
- **企业内部培训**：改变企业文化
- **建立互助小组**：社区支持网络
- **课程改革**：推动教育内容改革
- **快速响应**：对歧视事件及时发声
- **资源整合**：整合各方力量

### 6. 技能冷却系统

- 所有主动技能有冷却时间（2-3回合）
- 使用后需要等待冷却才能再次使用
- 增加了资源管理的策略深度
- 玩家需要在"现在使用"和"等待更好时机"之间权衡

### 7. 被动技能触发机制

- 被动技能会在特定条件下自动触发
- 例如：企业领袖在职场赚取资金时额外+1
- 例如：教育工作者在教育象限行动后50%概率抽牌
- 增加了角色的独特性和游戏的随机性

## 🎮 游戏机制改进总结

### 复杂度提升
| 机制 | 旧版本 | 新版本 |
|------|--------|--------|
| 角色技能 | 简单描述 | 主动+被动+冷却 |
| 进度系统 | 单一总进度 | 4个象限+总进度 |
| 事件处理 | 即时效果 | 持续+失败惩罚 |
| 项目系统 | 一次性投入 | 累计贡献+永久效果 |
| 卡牌数量 | 12张行动卡 | 20张行动卡 |
| 项目数量 | 6个项目 | 8个项目 |
| 事件数量 | 20张事件 | 25张事件 |

### 策略深度增加

1. **长期规划**：项目需要多回合协作，需要提前规划
2. **时机把握**：技能冷却要求玩家选择最佳使用时机
3. **领域平衡**：四个象限都需要关注，不能偏废
4. **合作激励**：永久效果鼓励玩家共同完成项目

### 教育价值提升

1. **系统性思维**：四个象限相互关联，体现社会复杂性
2. **长期视角**：项目累计机制体现社会变革需要持续投入
3. **真实议题**：新增事件关注AI偏见、母职惩罚等前沿话题
4. **协作共赢**：永久效果机制强化"平等让所有人受益"的理念

## 📊 游戏平衡性

新版本的平衡性考虑：

1. **角色平衡**：
   - 政策制定者和教育工作者：高影响力，适合破解刻板印象
   - 企业领袖：高资金，适合启动项目
   - 社区组织者：高支持+额外手牌，灵活性最强

2. **难度曲线**：
   - 象限进度系统使游戏更具挑战性
   - 事件失败惩罚增加紧迫感
   - 永久效果提供正向反馈，避免后期乏力

3. **游戏时长**：
   - 预计45-75分钟（视玩家数量和策略）
   - 项目累计机制会适当延长游戏时长
   - 但永久效果会加速后期进程
